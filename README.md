# ChatbotChambers

Two chatbots talk to each other while the user watches.

## Local setup

1. Copy `/home/runner/work/ChatbotChambers/ChatbotChambers/.env.example` to `.env`.
2. Fill in any values you need, especially `OPENROUTER_API_KEY`.
3. Start the app with `/home/runner/work/ChatbotChambers/ChatbotChambers/run.sh`, or run the backend and frontend separately.
